# Button → Route Documentation

Each page below traces every interactive control (button / link / form)
to the route or function it triggers.

- `part1-layouts-static` — Navbar, Footer, About, Terms, Contact
- `part2-home-products` — Home, CategoryPage, ProductPage
- `part3-shops-cart-checkout` — Cart (incl. guest cart), Checkout, Invoice
- `part4-orders-account-wishlist` — Profile, Favorites, order history
- `part5-auth` — Login, Signup, Logout, Verify
- `part6-trader` — Trader dashboard + product CRUD + discounts
- `part7-emails` — Welcome / receipt mailers
- `part8-profile-components` — Shared components (toasts, modals, context)
